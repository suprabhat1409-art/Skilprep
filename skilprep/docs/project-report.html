SkilPrep — Project Synopsis & Developer Report

Synopsis

SkilPrep is a multi-discipline challenge and learning platform that supports coding, mathematics, science, logic, and general-knowledge exercises. It provides problem authoring, in-browser code editing and execution, automatic and manual evaluation, community tests, shared resources, and instructor (admin) workflows for review and feedback.

Architecture

  • Client: React + Vite SPA (dev server at http://localhost:5173/).
  • Server/API: Node.js + Express providing REST endpoints under server/src.
  • Database: MongoDB with Mongoose models for Users, Problems, Submissions, Communities, Tests, Resources.
  • Sandbox & Evaluators: Modular evaluators in server/src/evaluators and a sandbox for safe code execution.
  • File uploads: handled via multer and stored in uploads/.

Tech Stack & Rationale

  • React + Vite: fast dev feedback and modern frontend toolchain.
  • Node + Express: lightweight REST API framework and large ecosystem.
  • MongoDB + Mongoose: flexible data models for documents like problems and submissions.
  • Monaco Editor: production-ready in-browser code editor for coding problems.
  • JWT + bcrypt: stateless auth plus secure password storage.
  • Evaluators: separate scoring logic per problem type makes system extensible and safer.

Development Workflow

  1. Configure environment: create server/.env and set MONGO_URI.
  2. Install dependencies for root, server, and client.
  3. Run seed data via npm run seed to populate demo users, problems, communities, and tests.
  4. Start both dev servers from project root: npm run dev (uses concurrently).
  5. Server changes are auto-reloaded by nodemon; client changes are hot-reloaded by Vite.

  • Routes: server/src/routes/ (auth, problems, community, tests, submissions).
